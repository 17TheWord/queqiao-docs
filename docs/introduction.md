# 鹊桥

✨ `Minecraft` 服务端 `Mod`/`Plugin`，实时接收玩家事件、`API` 广播消息 ✨

## 项目简介

鹊桥是一个适用于多种 `Minecraft` 服务端的 `Mod`/`Plugin`，能够实时分发玩家事件，并通过 `WebSocket` 与外部系统进行消息互通。

支持多种服务端类型，适合群聊、机器人、跨平台消息同步、甚至是LLM等场景。

## 主要功能

- **玩家事件分发**：实时推送玩家加入、离开、聊天、命令、成就、死亡等事件。
- **API 消息广播**：支持广播、私聊、标题、动画栏等多种消息接口。
- **多服务端支持**：`Vanilla`、`Spigot`、`Paper`、`Forge`、`Fabric`、`NeoForge`、`Folia`、`Velocity` 等多种服务端类型。
- **多版本支持**：整体生态支持 1.7.10 至 最新版本，未支持的版本可提交 Issue 或自行编译。
- **权限管理**：提供权限节点，支持权限插件进行管理。
- **命令交互**：通过 Rcon 实现服务端命令下发与反馈。
